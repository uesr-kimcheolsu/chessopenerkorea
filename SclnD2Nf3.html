모든 시실리안 디펜스 게임 중 52%가 2.Nf3를 둡니다.

흑에게 추천되는 세 가지 수에 대한 백의 최선의 수는 3.d4 cxd4 4.Nxd4라고 하며, 이 경우를 모두 통칭하여 '오픈 시실리안'이라고 부릅니다. 그리고 오픈 시실리안을 제외한 사이드라인을 총칭하여 '안티 시실리안'이라고 부르죠. 그런데 오픈 시실리안이 안티 시실리안 전체보다 더 많이 연구되었고, 더 많이 플레이됩니다. 오픈 시실리안의 하위 분류인 시실리안 나이도프가 오픈 게임만큼이나 이론이 깊다고 하니 말 다 했죠.

백이 오픈 시실리안으로 들어가기를 기대하고 흑이 별로인 수를 둘 때가 있는데, 이러면 백은 안티 시실리안을 골라서 흑의 2번째 수를 쓸모없게 만듭니다.

오픈 시실리안에서 흑은 4/5번째 수로 하위분류를 확정짓습니다.

  • d6 + Nf6(Nc3) + a6 = 나이도프(Najdorf)
  • d6 + Nf6(Nc3) + Nc6 = 클래시컬(Classical)
  • d6 + Nf6(Nc3) + e6 = 스헤베닝언(Scheveningen)
  • d6 + Nf6(Nc3) + e5 = 오픈-: 베니스 어택(Open-: Venice Attack)
  • d6 + Nf6(Nc3) + g6 = 드래곤(Dragon)
  • d6 + Nf6(Nc3) + Bd7 = 쿠프레치크(Kupreichik)
  • Nc6 + d5 = 님조 아메리칸(Nimzo-American)
  • Nc6 + e5 = 칼라시니코프(Kalashnikov)
  • Nc6 + Nf6(Nc3) + e5 = 스베시니코프(Sveshnikov)
  • Nc6 + g6 = 액셀러레이티드 드래곤(Accelerated Dragon)
  • Nc6 + Nf6(Nc3) + g6 = 하이퍼액셀러레이티드 드래곤: 피앙케토 바리에이션 (Hyperaccelerated Dragon: Fianchetto Variation)
  • e6 + a6 = 칸(Kan)
  • e6 + Nc6 = 타이마노프(Taimanov)
  • e6 + Nf6(Nc3) + Nc6 = 포 나이트(Four Knights)
